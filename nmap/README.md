# Nmap scripts


use it with <strong> sudo </strong>\
or as <strong>root</strong>

#

### scanning script - scan.sh

usage :
```bash
sudo bash scan.sh
```

example :
```bash
# Enter your IP
192.168.1.1
# Enter your Port
1-1337
# Select a Scan 
#  1 for Aggresive 
#  2 for Syn 
#  3 for TCP 
3
```
